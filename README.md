# miranaanwar.com

The personal website of [miranaanwar.com](https://miranaanwar.com/) built with Next.js 15 and React 18, featuring an editorial newspaper.

![Website Preview](public/assets/website_preview.png)

## ✦ set-up

1. Install the dependencies

```
npm install
```

2. Start the development server

```
npm run dev
```
